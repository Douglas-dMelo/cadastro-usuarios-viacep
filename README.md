# Cadastro de Usuário

Projeto simples de cadastro de usuários usando HTML, CSS e JavaScript.

## Funcionalidades

- Preenchimento automático do endereço a partir do CEP (ViaCEP)
- Salvamento dos dados durante a sessão do navegador (sessionStorage)
- Validação de formulário
- Tema claro e escuro

## Tecnologias

- HTML
- CSS
- JavaScript
- Fetch API
- ViaCEP
